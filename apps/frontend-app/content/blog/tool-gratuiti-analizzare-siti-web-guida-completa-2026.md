## Tool per Performance e Velocità {#performance}

La velocità è il fattore più critico per l'esperienza utente e il posizionamento SEO.

### 1. Google PageSpeed Insights

**URL**: [pagespeed.web.dev](https://pagespeed.web.dev)

**Cosa analizza:**
- Core Web Vitals (LCP, FID, CLS)
- Performance score 0-100
- Suggerimenti di ottimizzazione specifici
- Dati reali da Chrome UX Report

**Come usarlo strategicamente:**
1. Inserisci l'URL del potenziale cliente
2. Salva lo score mobile (più importante del desktop)
3. Identifica i 3 problemi principali
4. Stima il tempo di implementazione

**Score di riferimento:**
- 0-49: Scarso (opportunità alta)
- 50-89: Migliorabile (opportunità media)
- 90-100: Buono (opportunità bassa)

### 2. GTmetrix

**URL**: [gtmetrix.com](https://gtmetrix.com)

**Cosa analizza:**
- Performance score
- Tempo di caricamento completo
- Dimensione pagina
- Numero di richieste HTTP
- Waterfall dettagliato

**Vantaggi rispetto a PageSpeed:**
- Report più dettagliati
- Storico delle analisi
- Test da diverse location
- Video del caricamento

### 3. WebPageTest

**URL**: [webpagetest.org](https://webpagetest.org)

**Cosa analizza:**
- Test multi-step (navigazione completa)
- Filmstrip visuale del caricamento
- Analisi comparativa A/B
- Test da 40+ location globali

**Ideale per:**
- Analisi approfondite
- Report tecnici dettagliati
- Benchmark competitivi

## Tool per Analisi SEO {#seo}

L'ottimizzazione SEO determina la visibilità organica sui motori di ricerca.

### 4. Google Search Console

**URL**: [search.google.com/search-console](https://search.google.com/search-console)

**Cosa analizza:**
- Indicizzazione pagine
- Errori di crawling
- Performance organica
- Mobile usability
- Core Web Vitals

**Limitazione:** Richiede verifica proprietà (utile solo per siti propri)

### 5. Screaming Frog SEO Spider (Free)

**URL**: [screamingfrog.co.uk](https://www.screamingfrog.co.uk/seo-spider/)

**Cosa analizza:**
- Tutti i meta tag (title, description)
- Struttura heading (H1, H2, H3)
- Link interni e esterni
- Immagini senza alt text
- Redirect e errori 404

**Versione gratuita:** Analizza fino a 500 URL

**Output ideale per proposte:**
- Lista completa errori SEO
- Prioritizzazione per impatto
- Stima effort di correzione

### 6. Ubersuggest (Free Tier)

**URL**: [neilpatel.com/ubersuggest](https://neilpatel.com/ubersuggest/)

**Cosa analizza:**
- Domain authority
- Backlink profile
- Keyword ranking
- Traffico stimato
- Competitor analysis

**Limitazioni free:** 3 ricerche/giorno

## Tool per Sicurezza e HTTPS {#sicurezza}

La sicurezza è un fattore di ranking e un requisito legale (GDPR).

### 7. SSL Labs Server Test

**URL**: [ssllabs.com/ssltest](https://www.ssllabs.com/ssltest/)

**Cosa analizza:**
- Validità certificato SSL
- Configurazione HTTPS
- Vulnerabilità note
- Rating A+ → F

**Perché è importante:**
- Chrome mostra "Non sicuro" senza HTTPS
- Google penalizza i siti HTTP
- GDPR richiede crittografia dati

### 8. Security Headers

**URL**: [securityheaders.com](https://securityheaders.com)

**Cosa analizza:**
- HTTP security headers
- Content Security Policy
- X-Frame-Options
- HSTS

**Score di riferimento:**
- A+/A: Configurazione ottimale
- B/C: Miglioramenti necessari
- D/F: Vulnerabilità critiche

### 9. Mozilla Observatory

**URL**: [observatory.mozilla.org](https://observatory.mozilla.org)

**Cosa analizza:**
- Best practice sicurezza
- Configurazione TLS
- Cookie security
- Subresource integrity

## Tool per Analisi Tecnologica {#tecnologia}

Identificare le tecnologie usate aiuta a valutare complessità e opportunità.

### 10. BuiltWith

**URL**: [builtwith.com](https://builtwith.com)

**Cosa identifica:**
- CMS (WordPress, Shopify, Wix, etc.)
- Framework JavaScript
- Analytics e tracking
- CDN e hosting
- Plugin e integrazioni

**Perché è utile:**
- Stima complessità migrazione
- Identifica tecnologie obsolete
- Valuta stack tecnologico

### 11. Wappalyzer

**URL**: [wappalyzer.com](https://www.wappalyzer.com)

**Disponibile come:** Estensione browser (Chrome, Firefox)

**Cosa identifica:**
- CMS e framework
- E-commerce platform
- Marketing tools
- Analytics installati

### 12. WhatRuns

**URL**: [whatruns.com](https://www.whatruns.com)

**Estensione browser** che mostra in tempo reale le tecnologie di ogni sito visitato.

## Tool per Accessibilità {#accessibilita}

L'accessibilità è un obbligo legale e un'opportunità di mercato.

### 13. WAVE Accessibility Tool

**URL**: [wave.webaim.org](https://wave.webaim.org)

**Cosa analizza:**
- Conformità WCAG 2.1
- Errori accessibilità
- Contrasto colori
- Alt text immagini
- Struttura heading

### 14. axe DevTools

**Estensione browser** per test accessibilità dettagliati.

**Cosa identifica:**
- Violazioni WCAG A/AA/AAA
- Priorità correzione
- Codice specifico da modificare

### 15. Lighthouse Accessibility

Integrato in Chrome DevTools, fornisce score accessibilità 0-100.

## Come Usare le Analisi per Trovare Clienti {#trovare-clienti}

Ecco la strategia in 5 step per trasformare le analisi in opportunità di business.

### Step 1: Identifica il Target

- PMI locali senza presenza digitale forte
- E-commerce con performance scadenti
- Professionisti con siti datati
- Aziende con competitor più veloci

### Step 2: Analisi Rapida (5 minuti)

1. **PageSpeed Insights** → Score performance
2. **SSL Labs** → Sicurezza HTTPS
3. **BuiltWith** → Stack tecnologico
4. **WAVE** → Accessibilità base

### Step 3: Documenta i Problemi

Crea un report sintetico con:
- Screenshot dei risultati
- 3-5 problemi principali
- Impatto sul business (conversioni, SEO, sicurezza)
- Stima investimento per risoluzione

### Step 4: Primo Contatto

Email o messaggio con:
- Analisi gratuita del loro sito
- 1-2 problemi critici identificati
- Proposta di call per approfondimento
- Link a case study simili

### Step 5: Proposta Commerciale

Basata su dati oggettivi:
- Benchmark vs competitor
- ROI stimato miglioramenti
- Timeline e investimento
- Garanzie sui risultati
